Human Computer Interaction Project 1

Using An Elevator

Sequence of Actions:

1) Press up or down elevator button
2) Elevator opens
3) User steps in
4) User chooses floor and presses floor button
5) (optional) User hits the close door button for faster service
6) Elevator moves to selected floor w/ display indicating current floor
7) Elevator stops and user exits

Common elevator usages:

- Avoiding stairs
- Taking groceries up
- Transporting materials on a cart
- Traveling with a large group of people
- Going up multiple floors
- Moving into a hotel/apartment
- Accessibility (wheelchair bound users)

Real Life Example

Location: CS and EOC Elevator

Issues with the current design:

- Asymmetric button layout
- Button order is not intuitive

- Could be more descriptive
- No sectioning of buttons
